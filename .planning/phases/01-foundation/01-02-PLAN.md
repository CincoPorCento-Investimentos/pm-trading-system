       - public enum CheckPriority { HIGH, MEDIUM, LOW }
       - Sem metodos extras — enum puro

    3. CREATE `CheckStep.java`:
       - public record CheckStep(String name, Status status, String detail, long durationMs)
       - Static factories:
         - `public static CheckStep ok(String name, long durationMs)` — status=OK, detail="" (step passou sem detalhes adicionais)
         - `public static CheckStep ok(String name, String detail, long durationMs)` — status=OK com detalhe

         - `public static CheckStep fail(String name, String detail, long durationMs)` — status=FAIL

    4. CREATE `CheckResult.java`:
       - public record CheckResult(String checkName, Status status, String message, long durationMs, java.time.Instant timestamp, java.util.List<CheckStep> steps)
       - Compact constructor: `{ steps = java.util.List.copyOf(steps); }` — garante imutabilidade
       - Static factories:
         - `public static CheckResult ok(String checkName, String message, long durationMs, List<CheckStep> steps)` — status=OK, timestamp=Instant.now()
         - `public static CheckResult fail(String checkName, String message, long durationMs, List<CheckStep> steps)` — status=FAIL, timestamp=Instant.now()
         - `public static CheckResult warn(String checkName, String message, long durationMs, List<CheckStep> steps)` — status=WARN, timestamp=Instant.now()
       - Query methods:
         - `public boolean isFailed()` — return status == Status.FAIL
         - `public boolean isWarning()` — return status == Status.WARN
         - `public boolean isOk()` — return status == Status.OK

    5. CREATE `SyntheticCheck.java`:
       - public interface SyntheticCheck
       - Methods (NAO annotations — evita reflection, per CONTEXT.md decisions):
         - `String getName()` — kebab-case, ex: "health-check"
         - `String getDescription()` — descricao humana
         - `String getGroup()` — agrupamento, ex: "infrastructure", "orders", "websocket"
         - `CheckPriority getPriority()` — HIGH, MEDIUM, LOW
         - `CheckResult execute()` — executa o check e retorna resultado
